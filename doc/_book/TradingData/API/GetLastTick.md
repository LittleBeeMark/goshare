# 2.2.2 Tick数据

* 从新浪财经相关页面获取的最新盘口价

* API路径 /v1/getLastTick

* 传参数,post方式
```js
{
    "Exchange":4,
    "Code":"601398"
}
```

* 返回数据
```js
{
	"success": true,
	"data": {
		"symbol": {
			"exchange": 4,
			"code": "601398"
		},

		"low": 5.74, // 最低价

	},

}
```
